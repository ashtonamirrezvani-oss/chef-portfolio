# Chef Sophia Lang - Professional Portfolio

Elegant, modern single-page website designed to help Chef Sophia Lang stand out when applying for chef positions. Built with Tailwind CSS for a clean, professional look that highlights her culinary education at Escoffier Culinary School in Boulder and her externship experience as a line cook at Sam's Italian Restaurant in Snowmass Village.

## Features
- **Hero Section**: Striking introduction with call-to-action
- **About Me**: Detailed bio and skills
- **Experience**: Highlighted education and externship
- **Signature Dishes**: 6 featured creations with image placeholders, descriptions, and boss/teacher testimonials
- **Menus Created**: 3 original menus she designed (with modal details)
- **Testimonials**: Quotes from mentors and employers
- **Gallery**: Additional visual placeholders
- **Contact**: Professional inquiry form

## How to Customize (Important!)
1. **Replace Name**: Search for "Sophia Lang" and replace with the actual chef's full name throughout the HTML.
2. **Add Real Photos**: 
   - Replace all `https://placehold.co/...` URLs with actual high-quality photos (chef portrait, plated dishes, kitchen action shots, menu scans).
   - Recommended: Upload photos to the repo's `images/` folder and update paths, or use free hosting like ImgBB.
   - For each placeholder, the text overlay tells exactly what photo to use (e.g., "Professional Chef Portrait").
3. **Update Bio & Details**: Edit the About section with personal story, specific achievements, or certifications.
4. **Real Testimonials**: Replace the sample quotes with actual feedback from Escoffier instructors and Sam's Italian Restaurant chefs/owners. Ask them for written references!
5. **Menus**: Update the menu items and descriptions to match real menus she created during school or externship. Add PDF links if available.
6. **Contact Info**: 
   - Update email in contact section and form action (recommend Formspree.io for free form handling).
   - Add real social media links (Instagram food account, LinkedIn).
7. **Resume**: Add a downloadable PDF link (upload resume.pdf to repo and link it).

## Deploy on GitHub Pages (Free Hosting)
1. Go to the repository on GitHub.
2. Settings > Pages
3. Source: Deploy from a branch > Branch: `main` > Folder: `/ (root)`
4. Save. Your site will be live at `https://ashtonamirrezvani-oss.github.io/chef-portfolio`
